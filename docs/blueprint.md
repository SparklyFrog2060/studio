# **App Name**: SmartSpace Planner

## Core Features:

- Room Creation: Allow users to create virtual rooms with customizable dimensions.
- Product Placement: Enable drag-and-drop functionality for adding smart home products to rooms.
- Product Catalog: Maintain a database of smart home products with details and pricing.
- Room Cost Estimation: Calculate the total cost of smart home products within each room.
- Total Cost Calculation: Provide a summary of the total cost for all rooms combined.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trustworthiness and technology. In dark mode: Dark Blue (#1E3A8A)
- Background color: Light gray (#F0F4F8) to provide a clean, modern backdrop. In dark mode: Dark gray (#121212)
- Accent color: Soft green (#81C784) for indicating savings and confirmations. In dark mode: Light Green (#A5D6A7)
- Font: 'Inter' sans-serif for both headings and body text, ensuring legibility and a modern feel.
- Use flat, minimalist icons to represent different smart home products and room types.
- Employ a grid-based layout for easy product placement and room organization.
- Use smooth transitions when adding products or creating rooms for enhanced UX.
- Support Polish language
